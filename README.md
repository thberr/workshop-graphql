# Lancer le projet

## Backend

```bash
cd backend/
```

```bash
python -m venv venv
```

```bash
source venv/bin/activate
```

```bash
pip install -r requirements.txt
```

```bash
uvicorn main:app --reload
```

Accéder à localhost:8000/graphql

## Frontend

```bash
cd frontend/
```

```bash
npm run dev
```

Accéer à `localhost:8000`
